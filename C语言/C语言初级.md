## C与语言5.0

初级指针：指针就是地址，指针就是指针变量

指针变量不同的访问权限不同，比如char*访问一个字节，int*访问4个字节

***<u>野指针</u>*** 

 1、int* p 指针未初始化

2、指针越界访问

3、指针指向的空间释放

```c
int* test()
{
	int num = 10;
	return &num;
}
int main()
{
	int* p = test();
	*p=200;//非法访问
	return 0;
}
```

规避野指针

1、指针初始化

2、小心指针越界

3、指针指向空间释放，及时置为NULL

4、避免返回局部变量的地址//或者说避免返回栈空间的地址

5、指针使用之前检查有
